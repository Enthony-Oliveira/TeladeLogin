### Login Steampunk — Front-end

Projeto front-end de tela de Login e Cadastro, desenvolvido com HTML, CSS e JavaScript, explorando um estilo visual steampunk.
O objetivo do projeto foi unir criatividade, identidade visual e fundamentos de front-end.

### Demo: https://telade-login.vercel.app/

### Sobre o projeto

Este projeto foi desenvolvido durante o curso técnico no CEEP, que incentiva a criatividade dos alunos na construção de interfaces visuais.
Para este trabalho, escolhi o tema steampunk, utilizando tons de marrom e elementos visuais como engrenagens, buscando criar ambientação e identidade própria para a interface.

###

### Funcionalidades

Tela de Login

Tela de Cadastro

Layout temático steampunk

Estrutura de formulário

Interações básicas com JavaScript

###

### Tecnologias utilizadas

HTML5

CSS3

JavaScript (Vanilla JS)

###

### Conceitos praticados

Estruturação semântica com HTML

Estilização e layout com CSS

Uso de imagem de fundo e paleta de cores

Manipulação de eventos com JavaScript

Organização visual e identidade temática
